 Custom 8-bit ALU in Verilog
🔹 Overview

This project implements a custom 8-bit ALU using Verilog with modular and structural design.

🔹 Features
Arithmetic operations:
Addition
Reverse subtraction
Increment
Logical operations:
NOT, AND, OR, NAND
Shift operations:
Arithmetic left/right shift
Rotation:
Rotate left/right
Comparison:
Set on equal
🔹 Flags
Zero flag
Negative flag
Overflow flag
🔹 Design Approach
Structural modeling for:
Adder
Multiplexers
Logic units
Behavioral modeling for control where needed
Reusable modular components
🔹 Testbench

Includes a complete testbench validating all ALU operations.
